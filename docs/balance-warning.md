## 余额预警规则
服务空间按量余额、实人认证、一键登录、短信、扩展存储这些计费产品余额预警规则如下:

### 服务空间按量余额

| 预警规则 | 提醒频率 |  提醒方式 |
| --- | --- | --- |
| 当系统预测您账户的可用余额可能不足以支撑未来 X 天消耗时，<br>系统将向您发送余额不足预警通知。 |一个自然日内（00:00 - 24:00）至多发送一次通知。<br>若连续2天无消耗记录，则停止提醒。  | 短信、邮件、公众号。<br>接收人为服务空间创建者。 |


```
注意：
 - 系统根据您的历史按量计费资源消费情况与当前账户可用余额进行预测，仅作为预测提醒。
 
 - 未来X天表示未来1天、3天、7天

```


### 实人认证、短信、一键登录

| 预警规则 | 提醒频率 |  提醒方式 |
| --- | --- | --- |
| 当系统预测您账户的可用余额可能不足以支撑未来 X 天消耗 或 余额小于X元时，<br>系统将向您发送余额不足预警通知。 |一个自然日内（00:00 - 24:00）至多发送一次通知。<br>若连续3天无使用记录，则停止提醒。  | 邮件、公众号。<br>接收人为服务开通者。 |


```
注意：
 - 系统根据您的历史使用记录情况与当前账户可用余额进行预测，仅作为预测提醒。
 
 - 未来X天表示未来7天、3天、1天
 - X元 表示 
       短信:50元、20元、10元
       一键登录:10元、3元、1元
       实人认证:50元、20元、10元

```

### 扩展存储

| 预警规则 | 提醒频率 |  提醒方式 |
| --- | --- | --- |
| 余额小于X元时，系统将向您发送余额不足预警通知。 |一个自然日内（00:00 - 24:00）至多发送一次通知。<br>若连续7天无使用记录，则停止提醒。  | 邮件、公众号。接收人为服务开通者。 |


```
注意：
- X元 表示50元、20元、10元 
```
