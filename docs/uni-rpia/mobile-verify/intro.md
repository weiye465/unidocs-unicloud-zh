# uni手机号实名核验

> 欢迎加入[uni手机号实名核验技术交流群](https://im.dcloud.net.cn/#/?joinGroup=65436862cc41b0763842cfc9)

uni手机号实名核验分为三个产品：
- 手机号二要素核验：核验用户的姓名、手机号码一致性
- 手机号三要素核验（简版）：核验用户的姓名、身份证号码、手机号码的一致性
- 手机号三要素核验（详版）：核验用户的姓名、身份证号码、手机号码的一致性，返回详细的不匹配原因；

## 应用场景

- 婚恋交友类：需要核实注册用户的真实姓名和在平台注册的手机号码及身份证号，提升交友的安全性。
- 金融保险类：用于核验相关人的身份信息，提升资金安全。
- 网约车、货运服务类：用于核验加盟人员身份，提升平台服务的安全性。

## 核验范围

目前，uni手机号实名核验支持范围如下：
- 支持携号转网，支持阿里宝卡、腾讯王卡等互联网公司与运营合作推出的手卡
- 只支持移动，电信，联通下发的手机主卡，广电号暂不支持
- 不支持物联网卡、虚拟运营商卡（如阿里小号）
